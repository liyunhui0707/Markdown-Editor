| 14.9 | Frontmatter visual fix | Done | Hybrid-cm6 detects a top-of-file YAML frontmatter region (strict `---` opening line + later strict `---` closing line) via a `detectFrontmatter(state)` helper, then a single early-return guard at the top of the syntax-tree iterator suppresses ALL decoration emission for nodes whose `from < frontmatter.to`.
